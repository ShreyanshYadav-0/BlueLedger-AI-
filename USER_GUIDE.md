- flask
- pandas
- scikit-learn
- matplotlib
- reportlab

----------------------------------------------------------------

# 2. INSTALLATION PROCESS

Step 1:
Install Python

Step 2:
Open project folder in VS Code

Step 3:
Install required libraries

Command:

pip install flask pandas scikit-learn matplotlib reportlab

Step 4:
Run application

Command:

python app.py

Step 5:
Open browser

URL:

http://127.0.0.1:5000
